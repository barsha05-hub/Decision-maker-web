# Smart Decision Helper

A web-based application that helps users make better decisions by
analyzing situations, options, pros & cons, and scoring outcomes.

## Features
- Multiple decision categories (Job, College, Business, Career, Personal)
- Smart scoring algorithm
- Auto-generated pros & cons
- Clean and responsive UI
- 100% English interface

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)
